Moon Animator - Roblox Plugin

ENG VERSION:
Install the *.rbxm file, open/create a play in Roblox Studio, place the installed file,  
then through the Explorer in the Studio you will see the Moon Animator folder. 
Open the "Plugins" tab at the top and drag this folder there.

Moon Animator on Market: https://create.roblox.com/store/asset/4725618216/Moon-Animator-2

RU VERSION:
Установите файл *.rbxm, откройте/создайте игру в Roblox Studio, поместите установленный файл,
затем через проводник в студии вы увидите папку Moon Animator. 
Откройте вкладку "Плагины" вверху и перетащите эту папку туда.

Moon Animator в Магазине: https://create.roblox.com/store/asset/4725618216/Moon-Animator-2

Moon Animator - Roblox Plugin

ENG VERSION:
Install the *.rbxm file, open/create a play in Roblox Studio, place the installed file,  
then through the Explorer in the Studio you will see the Moon Animator folder. 
Open the "Plugins" tab at the top and drag this folder there.

Moon Animator (after installation, you will have it in stock) | the link is fixed at the very bottom

RU VERSION:
Установите файл *.rbxm, откройте/создайте игру в Roblox Studio, поместите установленный файл,
затем через Exporer в Роблокс Студио вы увидите модель "Moon Animator".
Откройте вкладку "Плагины" вверху и перетащите эту папку туда.
*или же нажмите по папке Правой Кнопкой Мыши (ПКМ), и выберите "Save as Local Plugin" (опция может называться иначе (в связи с обновлениям в Роблокс Студио), но исход остается тем же)

Moon Animator (после установки, он будет у Вас в наличии) | Ссылка закреплена в самом низу
